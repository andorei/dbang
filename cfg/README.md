# dbang Configuration Files

By convention config file name starts with the name of the dbang utility to which it relates:

* ddiff_<...>.py - ddiff config file
* dtest_<...>.py - dtest config file
* dget_<...>.py - dget config file
* dput_<...>.py - dput config file
* hedwig_<...>.py - hedwig config file
* dfifo_<...>.py - dfifo config file

Copy `sample_sources.py` to `sources.py` and specify your data sources in the latter.
